Sistema UFC

Sistema em Python para cadastrar e gerenciar alunos da UFC.
Permite criar contas, fazer login, trocar/trancar curso, virar monitor e registrar horas.

Arquivos
aluno.py   - Classe Aluno e Monitor
campus.py  - Classe Campus
cursos.py  - Lista de campus e cursos
main.py    - Menu principal do sistema

* Funções principais *

- Cadastro de aluno (com validação de CPF)

- Login com usuário e senha

- Ver curso

- Trocar de curso

- Trancar curso

- Deletar conta

- Sistema de monitoria (virar monitor e registrar horas)


Como executar

No terminal:
python main.py


Requisitos:
Python 3.8 ou superior


Observações:
Sistema totalmente no terminal
Monitoria soma horas à carga obrigatór
